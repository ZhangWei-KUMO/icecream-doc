#### 发布周期

- 修订版本号：每周末会进行日常`bug`更新。（如果有紧急的`bug`，则任何时候都可发布）
- 次版本号：每月发布一个带有新功能的向下兼容的版本。
- 主版本号：含有破坏性更新和新功能，不在发布周期内。
